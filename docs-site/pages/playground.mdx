### Using components

With Go1d you won't need to write much CSS if any at all as all the customization is controlled via props on the components.

The props on the components will ensure consistent spacing, sizing, and colors across all Go1 applications.

#### View component

The most fundamental component for building a UI is the [View](https://go1d.go1.com/storybook/?path=/docs/view--basic-usage) Component. The View component uses flex box to define the layout.

```react-inline
<View mode="light" color="noteLow">
  Use the View component where you would usually use a div.
</View>
```

#### Text component

All Text should be wrapped in a [Text](https://go1d.go1.com/storybook/?path=/docs/text--unstyled-text) Component or a specific typography component, such as the Heading component. The [Text](https://go1d.go1.com/storybook/?path=/docs/text--unstyled-text) component handles font and includes some responsive capabilities out of the box.

```react-inline
<Heading color="default">Edit the code snippet to change this Heading</Heading>
```

```react-inline
<Text
  element="p"
  color="default"
  fontSize={3}
  fontWeight="normal"
  fontFamily="sansSerif"
  lineHeight="paragraph"
>
Edit the code snippet to edit this paragraph using the Text component.
</Text>
```

#### Colors

When specifying colors in Go1d components, use the names listed on the [Colors](https://go1d.go1.com/storybook/?path=/docs/color) page.

This will ensure all colors are consistent across Go1 applications as well as any tweaks or changes update everywhere.

```react-inline

<>
<Text element="p" lineHeight="paragraph" color="accent">Accent</Text>
<Text element="p" lineHeight="paragraph" color="contrast">Contrast</Text>
<Text element="p" lineHeight="paragraph" color="default">Default</Text>
<Text element="p" lineHeight="paragraph" color="subtle">Subtle</Text>
<Text element="p" lineHeight="paragraph" color="successLow">Success Low</Text>
<Text element="p" lineHeight="paragraph" color="noteLow">Note Low</Text>
<Text element="p" lineHeight="paragraph" color="warningLow">Warning Low</Text>
<Text element="p" lineHeight="paragraph" color="dangerLow">Danger Low</Text>
</>
```
