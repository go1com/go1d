A component that visually hides content so that you may provide screen readers with meaningful text.

## Examples

### Visually Hidden

```.jsx
<VisuallyHidden>
  Loading courses
</VisuallyHidden>
```

## Props

```!jsx
<ComponentDoc component="VisuallyHidden" />
```
