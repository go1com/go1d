A generic component for display content type


### Base
```.jsx
<ContentType
  type="Course" />
```

### With text
```.jsx
<ContentType
  type="Course"
  text="Course"
 />
```

### With background
```.jsx
<ContentType
  type="Event"
  text="event"
  background="default"
  width={70}
 />
```
