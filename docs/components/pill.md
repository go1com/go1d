## Examples

### Default pill

```.jsx
<Pill>Default</Pill>
```

### Accent pill

```.jsx
<Pill color='accent'>Accent</Pill>
```

### Success pill

```.jsx
<Pill color='success'>Success</Pill>
```

### Note pill

```.jsx
<Pill color='note'>Note</Pill>
```

### Warning pill

```.jsx
<Pill color='warning'>Warning</Pill>
```

### Danger pill

```.jsx
<Pill color='danger'>Danger</Pill>
```
