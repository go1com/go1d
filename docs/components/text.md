All text in our apps should be wrapped in a Text component.

Try out the props below.


### Base
```.jsx
<Text
  color='subtle'
  fontSize={4}
  fontWeight="semibold"
  lineHeight="paragraph"
>
  My large semibold subtle text
</Text>
```
